## Installation

1- Rename file from `rename_local.properties`  to  `local.properties`
<br/>
<br/>
2- Rename file from `rename-google-services.json`  to  `google-services.json`
<br/>
<br/>
3- Build project to generate needed files
<br/>
<br/>
4- Chose Build Variant
<br/>
<br/>
5- Run!
